# PixeArt

Turn any image into beautiful pixel art mosaics.

![Screenshot](screenshots/app.png)

## Features

-  Image Preview
-  Adjustable Dot Size
-  Progress Bar
-  PNG/JPG/WebP Export
-  Native macOS App

## Installation

### macOS

Download the latest release from Releases.

### Source

```bash
pip install -r requirements.txt
python main.py
```
