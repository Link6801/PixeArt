# PixeArt

Turn any image into beautiful pixel art mosaics.

## Screenshots

### Main Interface

![Main Interface](screenshots/GUI.png)

## Features

-  Image Preview
-  Adjustable Dot Size
-  Progress Bar
-  PNG/JPG/WebP Export
-  Native macOS App

## Installation

### macOS

Download the latest release from Releases.

### Source

```bash
pip install -r requirements.txt
python main.py
```
